## Notes

- **112** is the pan-European emergency number and works in all countries in this table.
- Countries marked `low` confidence either have no known dedicated poison centre, or only ambulance/ER referral exists.
- **Public-callable?** `yes` means the general public can call; `no` means the line is for healthcare professionals only.
- **NL (Netherlands):** NVIC is clinician-only; public should call 112 or GP out-of-hours service.
- **GB (UK):** NPIS does not take public calls. Public should call 111 (England/Wales) or 999 for emergencies.
- **IE (Ireland):** Public line hours are 8am–10pm; call 112/999 at night for emergencies.
- **MT (Malta):** Public line hours 08:00–20:00; call 112 outside hours.
- **FR (France):** Different CAP centres serve different regions; any CAP can help nationwide.
- **DE (Germany):** Each state has a Giftnotruf; all use the regional 19240 suffix (e.g. 030 19240 Berlin). BfR coordinates.
- **RU (Russia) / UA (Ukraine) / BY (Belarus):** Limited current data; advise calling 103/112 for ambulance.
- **Kosovo (XK):** Not a UN member state; ISO 3166 code used is XK (user-assigned).
